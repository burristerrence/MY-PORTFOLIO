Police Innovation image (from your PC)

To use your own picture from your Pictures folder (or "picture" folder):

1. Open that folder on your PC and find the image you want.
2. Copy the image (Ctrl+C or right-click > Copy).
3. Open this folder in File Explorer:
   MY PORTFOLIO\public\
4. Paste the image here (Ctrl+V).
5. Rename the pasted file to:  police-innovation.jpg
   (If your file is .png, rename to police-innovation.png and update the project code to use .png.)

The portfolio will then show your image on the Police Innovation card.
